# ✨ TaskZen

TaskZen is a futuristic productivity dashboard built using HTML, CSS, and JavaScript.

It combines task management with a modern dashboard experience featuring live productivity stats, animated visuals, motivational quotes, and a clean glassmorphism UI.

## Live Demo

---

# 🚀 Features

* Add, Complete & Delete Tasks
* Task Filtering (All / Completed / Pending)
* Live Digital Clock
* Dynamic Motivational Quotes
* Dark / Light Theme Toggle
* Animated Progress Bar
* Productivity Statistics Dashboard
* Interactive Doughnut Chart
* Animated Particle Background
* Local Storage Support
* Fully Responsive Design
* Modern Glassmorphism UI

---

# 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Chart.js
* tsParticles
* Google Fonts

# 🎨 UI Overview

TaskZen uses a dashboard-style layout:

* Left Panel → Motivational Quotes & Live Clock
* Center Panel → Main Task Manager
* Right Panel → Productivity Stats & Progress Graph

The UI is inspired by modern productivity and SaaS applications.
